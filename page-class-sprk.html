Learn with Your Robot Friend

You are going to spend the entire camp with your robot friend — SPRK. SPRK is smart and understands your commands. Your job is to learn how to program the commands in order to let SPRK accomplish challenging tasks.

Learn More about SPRK »

Approachable Programming

Beginners can give robots commands by drawing a path in the app for their robot to follow. Intermediate coders can utilize the familiar block-based drag and drop interface, while pros can use new text programming and write their own code.

Simple Examples Show Big Ideas

SPRK uses a simple and visual programming language to build program blocks and control robots. However, the simple blocks reveal the most important concepts about computer programming such as variable, conditions, and loops.

Learn Programming with Fun

Through connected play and discovery, the robots are challenging and inspiring brilliant young minds. From math and science to programming and the arts, hands-on activities expand imaginations and help shape a brighter future.

Hands-On Learning Experiences

We focus on hands-on problem solving skills and techniques. You will learn how to solve problems using different programming methods. Everybody competes with each other, but also collaborate and work with each other to accomplish the challenging tasks together.
