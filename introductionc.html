Coding Minds的同学
如何脱颖而出

面对计算机时代的同学，我们设计课程的宗旨不止是：动手、创新、有用、有趣，还要带领同学们做自己的项目，提升编程的能力，为竞赛打基础，为升学做准备。让学习计算机真正地帮助到同学申请到好的大学。

在竞赛中锻炼与提高

Coding Minds的每一个同学平均每年至少要参加2次计算机和科学竞赛 竞赛在给同学们提供目标，开阔眼界的同时，也帮助同学们锻炼综合素质，获得名校申请的信心

竞赛为同学学习编程 完善项目提供了目标和动力
在竞赛中 不止是编程能力得到提升 综合能力也能得到锻炼
竞赛成绩作为荣誉和成就 可以为名校申请提供有力的支持
Coding Minds的同学平均每年参加两次美国顶级编程竞赛
不断挑战科研与创新

Coding Minds同学的科研方向涵盖人工智能，大数据，移动端开发，算法，物联网，网络服务等多个领域。Coding Minds的同学最早在5年级就可以熟练掌握运用人工智能与神经网络，8年级便可以进行科研试验并发表国际会议论文

Coding Minds为同学提供来自美国顶尖高校及著名公司的科研导师 保证同学接受最顶级的科研指导
研究方向主要以同学兴趣为主 找出合适的切入点 给同学搭建实现自己理想的舞台
研究实验 发表论文的过程充满了挑战 Coding Minds总结出独有的一对一科研指导方法 帮助同学们踏上科研道路
Coding Minds的同学最早8年级就可以人工智能领域期刊发表文章 多名同学因科研项目而步入名校
创业从青少年开始

Coding Minds鼓励同学从头开始，用完整的流程实践创业的全过程。带给同学不只是技术上的进步，更有丰富的履历和综合素质的提升。

创业首先需要好的想法 Coding Minds从生活出发 启发同学想出真正切合实际的创业点子
实现想法需要技术支持 Coding Minds为同学准备了多门实用性极强的开发课程 保证同学能出色的完成研发
创业离不开商业知识 Coding Minds利用与业界的紧密联系 为同学提供最专业的商业课程
申请专利 成立公司 寻求投资 同学项目完成以后 Coding Minds会帮助同学申请专利 成立创业公司 召开投资展示会 完成整个创业流程
发布作品 保留记录

日日奋笔，华章自成。Coding Minds要求同学发布并保留自己的每一个项目，记录自己成长的同时，为以后的升学做好准备，提供素材。

编程作品是学习编程的成果反映 Coding Minds要求同学每一门课都要做出自己的编程作品
作品发布渠道同样重要 Coding Minds同学的编程大量发布在移动端 网络端 游戏客户端等 保证流量和知名度
开源作品发布有利于同学同好之间互相交流 取长补短 不断完善
发布后的作品作为升学时的有力支持 在名校申请当中发挥重要作用
志愿服务 回馈社会

十年树木，百年树人。Coding Minds为同学提供科技相关的志愿者服务机会，培养领导力的同时启发同学服务社会，回馈社会。

服务意识和反馈思想是一个科技创业工作者所需要具备的基本素质 也是未来社会看重的道德品质
好的志愿服务活动可以给同学带来更多 Coding Minds为同学提供科技相关志愿者服务平台 服务社会的同时自身也得到锻炼
Coding Minds组织的志愿活动可以接触到很多有知名度的专家学者 行业精英 与他们一起工作会使同学收益良多
志愿服务在回馈社会 提升自我的同时 也为大学申请积累了雄厚的资本
